How to Play Snake by Danny

Use the arrow keys to move the snake.
The goal is to eat the food and grow the snake.
If the snake runs itself, the game is over. you can restart
the game by clicking the restart button. also, you can press the
space bar to pause the game.

  • Move-left - left arrow '<' key
  • Move-Right - Right arrow '>' key
  • Move-Down - Down arrow key
  • Move-Up - Up arrow key
  • Pause Game- space-bar key
  • Restart/Replay - Restart button
